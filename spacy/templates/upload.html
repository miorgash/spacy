Sudachi Tokenizer

使い方

csv を選択して Submit するとトークナイズの結果が JSON 形式でダウンロードされます．

入力形式

2 列の csv のみ処理可能．

1 列目：id

2 列目：テキスト



例

id, value

1, "舞台藝術が好きなんです。"

2, "初音ミクは歌が上手です。"

出力形式

spaCy の API Reference（https://spacy.io/api/token#attributes）参照



{

    id : [

        [

            表層形（Token.text）,

            見出語（Token.lemma_）,

            品詞（全言語共通のもの）（Token.pos_）,

            品詞（日本語のもの）（Token.tag_）,

            依存関係 # Token.dep_

        ]

}
